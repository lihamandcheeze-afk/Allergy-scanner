# Allergy Scanner

## Setup Instructions

To set up the Allergy Scanner app, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/lihamandcheeze-afk/Allergy-scanner.git
   ```
2. Change into the project directory:
   ```bash
   cd Allergy-scanner
   ```
3. Open the project in Xcode.
4. Run the app on a simulator or a device that has camera access enabled.

Make sure to grant camera permissions when prompted.
